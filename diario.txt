Eu aprendi alguns comandos essenciais que me ajudam a organizar projetos de forma fácil e eficiente. O git init cria um lugar para guardar as versões do seu código, git clone copia um projeto de outro lugar para o seu computador, git status mostra o que mudou no seu projeto, git add marca um arquivo que você mudou para ser guardado, git commit -m "[mensagem]" guarda as mudanças com uma descrição do que você fez, git push envia suas mudanças para um lugar online, git diff mostra as diferenças exatas entre o que você mudou e a última versão guardada, git log mostra todo o histórico de "guardadas" (commits) que você fez.
Isso ajuda a ter o histórico do seu trabalho organizado.
